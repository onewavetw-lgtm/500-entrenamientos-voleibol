# Landing Page - Voleibol (+500 Entrenamientos)

Esta es una landing page estática responsiva preparada para deploy en Vercel sin erros 404 de imágenes.

## Estructura de código
```
/
├── index.html
├── styles.css
├── script.js
└── assets/
    ├── hero-mockup.png
    ├── pagina-1.png
    ├── pagina-2.png
    ├── pagina-3.png
    ├── pagina-4.png
    ├── pagina-5.png
    ├── bonus-1.png
    ├── bonus-2.png
    ├── bonus-3.png
    ├── plano-basico.png
    └── plano-completo.png
```

## Instrucciones para subir a Vercel
1. Mantén todas las imágenes **exclusivamente dentro de la carpeta `assets/`**.
2. Los nombres de los archivos **deben ser 100% exactos a la lista de arriba** (sin acentos, sin espacios, sin mayúsculas).
3. Sube todos los archivos (index.html, styles.css, script.js y la carpeta assets completa) a un repositorio en **GitHub**.
4. En Vercel, iniciar sesión y hacer clic en **"Add New Project"**.
5. Importar el repositorio desde tu cuenta de GitHub.
6. En la configuración final del proyecto en Vercel, asegúrate de mantener:
   - **Framework Preset**: `Other`
   - **Build Command**: `(vacío)`
   - **Output Directory**: `(vacío o /)`
7. Hacer clic en **Deploy**. ¡Listo!
